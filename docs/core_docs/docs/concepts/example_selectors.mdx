## 概述

为了获得更好的效果，一种常见的提示技术是将示例包含在提示中。这被称为[少量示例提示](/docs/concepts/few_shot_prompting/)。

这为[语言模型](/docs/concepts/chat_models/)提供了其应如何运行的具体示例。有时这些示例会被硬编码到提示中，但在更高级的场景中，动态地选择这些示例会更合适。

**示例选择器** 是负责选择示例并将其格式化为提示的类。

> 请参阅 [API 参考文档以获取更多信息](https://v03.api.js.langchain.com/classes/_langchain_core.example_selectors.BaseExampleSelector.html)
