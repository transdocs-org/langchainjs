# 模型认证失败

你的模型提供商拒绝了你访问其服务的权限。

## 故障排查

以下方法可能有助于解决此错误：

- 确认你的 API 密钥或其他凭据是正确的。
- 如果你依赖环境变量进行身份验证，请确认变量名称正确且已设置值。
  - 注意某些环境（如 Cloudflare Workers）不支持环境变量。
  - 对于某些模型，你可以尝试显式传递 `apiKey` 参数，以排除此类环境变量问题：

```ts
const model = new ChatOpenAI({
  model: "MODEL_NAME",
  apiKey: "YOUR_KEY_HERE",
});
```

- 如果你使用了代理或其它自定义端点，请确保你的自定义提供商不期望使用其他的身份验证方案。
