# 简介

**LangChain** 是一个用于开发由大型语言模型（LLMs）驱动的应用程序的框架。

LangChain 简化了 LLM 应用生命周期的每一个阶段：

- **开发**：使用 LangChain 的开源 [构建模块](/docs/concepts/lcel)、[组件](/docs/concepts) 和 [第三方集成](/docs/integrations/platforms/) 构建你的应用程序。
  使用 [LangGraph.js](/docs/concepts/architecture#langgraph) 构建具有一流流式传输和人工介入支持的状态化代理。
- **生产化**：使用 [LangSmith](https://docs.smith.langchain.com/) 检查、监控和评估你的链，以便你可以持续优化并自信地部署。
- **部署**：使用 [LangGraph Cloud](https://langchain-ai.github.io/langgraph/cloud/) 将你的 LangGraph 应用程序转化为生产就绪的 API 和助手。

具体来说，该框架包括以下开源库：

- **`@langchain/core`**：基础抽象和 LangChain 表达式语言。
- **`@langchain/community`**：第三方集成。
  - 合作伙伴包（例如 **`@langchain/openai`**、**`@langchain/anthropic`** 等）：一些集成进一步拆分为仅依赖 **`@langchain/core`** 的轻量级独立包。
- **`langchain`**：组成应用程序认知架构的链、代理和检索策略。
- **[LangGraph.js](https://langchain-ai.github.io/langgraphjs/)**：通过将步骤建模为图中的边和节点，使用 LLM 构建强大且具有状态的多角色应用程序。
- **[LangSmith](https://docs.smith.langchain.com)**：一个开发者平台，让你能够调试、测试、评估和监控 LLM 应用程序。

:::note

这些文档专注于 JavaScript 版本的 LangChain 库。有关 Python 版本的 LangChain 库文档，请访问 [这里](https://python.langchain.com)。

:::

## 生态系统

### [🦜🛠️ LangSmith](https://docs.smith.langchain.com)

追踪并评估你的语言模型应用程序和智能代理，帮助你从原型过渡到生产环境。

### [🦜🕸️ LangGraph](https://langchain-ai.github.io/langgraphjs/)

使用 LLM 构建具有状态的多角色应用程序。与 LangChain 无缝集成，但也可以独立使用。LangGraph 支持生产级别的代理，已被 LinkedIn、Uber、Klarna、GitLab 等众多公司所信赖。
