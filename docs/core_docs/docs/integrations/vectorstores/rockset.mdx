# Rockset

Rockset（已被 OpenAI 收购）是一个在云端运行的实时分析型 SQL 数据库。
Rockset 提供以 SQL 函数形式实现的向量搜索功能，用于支持依赖文本相似性的 AI 应用程序。

## 安装配置

安装 rockset 客户端。

```bash
yarn add @rockset/client
```
