# Turbopuffer

## 配置

首先，您需要在此处注册一个 Turbopuffer 账户：[这里](https://turbopuffer.com/join)。  
注册完成后，您可以创建一个 API 密钥。

将您的 API 密钥设置为环境变量：

```bash
export TURBOPUFFER_API_KEY=<YOUR_API_KEY>
```
