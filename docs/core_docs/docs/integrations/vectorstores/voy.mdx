# Voy

[Voy](https://github.com/tantaraio/voy) 是一个使用 Rust 编写的 WASM 向量相似性搜索引擎。
它支持在非 Node 环境（如浏览器）中运行。您可以将 Voy 与 LangChain.js 一起用作向量存储。
