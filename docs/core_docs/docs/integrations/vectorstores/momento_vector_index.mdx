# Momento Vector Index (MVI)

[MVI](https://gomomento.com)：您数据的最高产、最易用的无服务器向量索引。要开始使用 MVI，只需注册一个账户即可。无需处理基础设施、管理服务器，也无需担心扩展问题。MVI 是一项可自动扩展以满足您需求的服务。无论是在 Node.js、浏览器还是边缘环境中，Momento 都能为您提供支持。

要注册并访问 MVI，请访问 [Momento 控制台](https://console.gomomento.com)。

## 配置

1. 在 [Momento 控制台](https://console.gomomento.com/)中注册一个 API 密钥。
2. 为您的环境安装 SDK。

   2.1 对于 **Node.js**：

   ```bash npm2yarn
   npm install @gomomento/sdk
   ```

   2.2 对于 **浏览器或边缘环境**：

   ```bash npm2yarn
   npm install @gomomento/sdk-web
   ```

3. 在运行代码前为 Momento 设置环境变量

   3.1 OpenAI

   ```bash
   export OPENAI_API_KEY=YOUR_OPENAI_API_KEY_HERE
   ```

   3.2 Momento

   ```bash
   export MOMENTO_API_KEY=YOUR_MOMENTO_API_KEY_HERE # https://console.gomomento.com
   ```
