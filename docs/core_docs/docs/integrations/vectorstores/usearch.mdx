# USearch

:::tip 兼容性
仅适用于 Node.js。
:::

[USearch](https://github.com/unum-cloud/usearch) 是一个用于高效进行相似性搜索和密集向量聚类的库。

## 安装设置

安装 [usearch](https://github.com/unum-cloud/usearch/tree/main/javascript) 包，这是一个为 [USearch](https://github.com/unum-cloud/usearch) 提供的 Node.js 绑定。

```bash npm2yarn
npm install -S usearch
```
