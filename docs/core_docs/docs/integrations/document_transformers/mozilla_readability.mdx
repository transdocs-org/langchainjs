# @mozilla/readability

在摄入 HTML 文档以供后续检索时，我们通常只关注网页的实际内容，而非语义结构。  
使用 MozillaReadabilityTransformer 从文档中剥离 HTML 标签可以生成内容更丰富的片段，从而提升检索效果。

## 安装配置

你需要安装 [`@mozilla/readability`](https://www.npmjs.com/package/@mozilla/readability) 和 [`jsdom`](https://www.npmjs.com/package/jsdom) 这两个 npm 包：

```bash npm2yarn
npm install @mozilla/readability jsdom
```

虽然对于该转换器本身而言不是必须的，但以下使用示例中需要用到 [`cheerio`](https://www.npmjs.com/package/cheerio) 进行网页抓取：

```bash npm2yarn
npm install cheerio
```
