# html-to-text

在摄入 HTML 文档以供后续检索时，我们通常只关注网页的实际内容，而非其语义结构。使用 `HtmlToTextTransformer` 剥离 HTML 标签可以使内容分块更加丰富，从而提高检索的有效性。

## 安装配置

你需要安装 [`html-to-text`](https://www.npmjs.com/package/html-to-text) npm 包：

```bash npm2yarn
npm install html-to-text
```

虽然 `HtmlToTextTransformer` 本身不需要额外依赖，但以下示例代码需要使用 [`cheerio`](https://www.npmjs.com/package/cheerio) 进行网页抓取：

```bash npm2yarn
npm install cheerio
```
