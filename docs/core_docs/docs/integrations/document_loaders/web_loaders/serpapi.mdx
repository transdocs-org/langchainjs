# SerpAPI 加载器

本指南展示了如何在 LangChain 中使用 SerpAPI 加载网页搜索结果。

## 概览

[SerpAPI](https://serpapi.com/) 是一个实时 API，可提供来自多个搜索引擎的搜索结果。它通常用于竞争对手分析和排名跟踪等任务。它使企业能够从所有搜索引擎的结果页面中抓取、提取和分析数据。

本指南展示了如何使用 LangChain 中的 `SerpAPILoader` 加载网页搜索结果。`SerpAPILoader` 简化了从 SerpAPI 加载和处理网页搜索结果的过程。

## 准备工作

你需要注册并获取你的 [SerpAPI API 密钥](https://serpapi.com/dashboard)。
