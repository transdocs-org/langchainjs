# Spider

[Spider](https://spider.cloud/?ref=langchainjs) 是[速度最快](https://github.com/spider-rs/spider/blob/main/benches/BENCHMARKS.md#benchmark-results)的网络爬虫。它可以将任意网站转换为纯 HTML、Markdown、元数据或文本，同时允许你使用 AI 执行自定义操作进行爬取。

## 概述

Spider 支持使用高性能代理防止被检测到，缓存 AI 操作，提供用于爬取状态的 Webhook、定时爬取等功能。

本指南展示了如何使用 [Spider](https://spider.cloud/) 爬取网站，并通过 LangChain 中的 `SpiderLoader` 加载适用于 LLM 的文档。

## 准备工作

在 [spider.cloud](https://spider.cloud/) 上获取你自己的 Spider API 密钥。
