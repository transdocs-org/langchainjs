# 文档加载器

[文档加载器](/docs/concepts/document_loaders) 会将数据加载为 LangChain 所需的格式，适用于 [检索增强生成 (RAG)](/docs/tutorials/rag) 等应用场景。

LangChain.js 从以下两个维度对文档加载器进行分类：

- [文件加载器](/docs/integrations/document_loaders/file_loaders/)，用于从本地文件系统中加载数据到 LangChain 格式。
- [网页加载器](/docs/integrations/document_loaders/web_loaders/)，用于从远程来源加载数据。

请参阅各分类的独立页面以了解更多内容。

:::info
如果你想编写自己的文档加载器，请参阅 [此指南](/docs/how_to/document_loader_custom/)。如果你想贡献一个集成实现，请参阅 [集成贡献指南](/docs/contributing)。
:::
