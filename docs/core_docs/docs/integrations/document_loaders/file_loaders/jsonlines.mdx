# JSONLines 文件

本示例介绍如何从 JSONLines（JSONL）文件加载数据。第二个参数是一个 JSONPointer，指向需要从文件中每个 JSON 对象提取的属性。文件中的每个 JSON 对象将生成一个文档。

示例 JSONLines 文件：

```json
{"html": "这是一个句子。"}
{"html": "这是另一个句子。"}
```

示例代码：

```typescript
import { JSONLinesLoader } from "langchain/document_loaders/fs/json";

const loader = new JSONLinesLoader(
  "src/document_loaders/example_data/example.jsonl",
  "/html"
);

const docs = await loader.load();
/*
[
  Document {
    "metadata": {
      "blobType": "application/jsonl+json",
      "line": 1,
      "source": "blob",
    },
    "pageContent": "这是一个句子。",
  },
  Document {
    "metadata": {
      "blobType": "application/jsonl+json",
      "line": 2,
      "source": "blob",
    },
    "pageContent": "这是另一个句子。",
  },
]
*/
```
