# Open AI Whisper 音频

:::tip 兼容性
仅支持 Node.js。
:::

本节介绍如何使用 [Open AI Whisper](https://platform.openai.com/docs/guides/speech-to-text) API 从音频文件加载文档对象。

## 环境准备

要运行此加载器，你需要在 Open AI 创建一个账户，并从 https://platform.openai.com/account 页面获取认证密钥。

## 使用方法

配置好认证密钥后，你可以使用该加载器创建音频转录内容，并将其转换为 Document 对象。
