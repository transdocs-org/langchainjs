# Discord 工具

Discord 工具使你的智能体具备搜索、读取和向 Discord 频道发送消息的能力。
当你需要与 Discord 频道进行交互时，该工具将非常有用。

## 准备工作

要使用 Discord 工具，你需要安装以下官方指定的同级依赖：

```bash npm2yarn
npm install discord.js
```
