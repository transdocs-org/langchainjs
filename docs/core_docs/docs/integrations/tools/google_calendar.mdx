# Google Calendar 工具

Google Calendar 工具允许你的代理从已连接的日历创建和查看 Google Calendar 事件。

## 安装配置

要使用 Google Calendar 工具，你需要安装以下官方的 peer dependency：

```bash npm2yarn
npm install googleapis
```
