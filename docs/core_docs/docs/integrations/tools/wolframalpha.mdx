# WolframAlpha 工具

WolframAlpha 工具可将您的智能体（agents）和链（chains）连接到 WolframAlpha 尖端的计算智能引擎。

## 准备工作

您需要从 [WolframAlpha 门户](https://developer.wolframalpha.com/) 创建一个应用并获取一个 `appid`。
