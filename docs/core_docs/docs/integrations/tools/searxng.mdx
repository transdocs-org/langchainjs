# Searxng 搜索工具

`SearxngSearch` 工具可将你的智能代理（agents）和链（chains）连接到互联网。

该工具封装了 SearxNG API，适用于通过 SearxNG API 执行元搜索引擎查询。它在回答与时事相关的问题时特别有用。
