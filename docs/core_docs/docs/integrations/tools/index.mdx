# 工具和工具包

import { CategoryTable, IndexTable } from "@theme/FeatureTables";

[工具](/docs/concepts/tools) 是专为模型调用而设计的实用程序：其输入旨在由模型生成，其输出则旨在返回给模型。

[工具包](/docs/concepts/tools/#toolkits) 是专为一起使用的工具集合。关于工具包集成的列表，请参阅[此页面](/docs/integrations/toolkits/)。

:::info
如果你想编写自己的工具，请参阅[此教程](/docs/how_to/custom_tools/)。如果你想贡献一个集成，请参阅[贡献集成](/docs/contributing)。
:::
