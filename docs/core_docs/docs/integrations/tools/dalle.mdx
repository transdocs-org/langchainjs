# Dall-E 工具

Dall-E 工具允许你的代理使用 OpenAI 的 Dall-E 图像生成工具来创建图像。

## 准备工作

你需要从 [OpenAI 网站](https://openai.com) 获取一个 OpenAI API 密钥，并将 `OPENAI_API_KEY` 环境变量设置为你刚刚创建的密钥。

要使用 Dall-E 工具，你需要安装 LangChain OpenAI 集成包：
