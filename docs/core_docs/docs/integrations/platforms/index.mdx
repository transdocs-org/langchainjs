# 提供者（Providers）

LangChain 与许多提供者进行了集成。

## 合作伙伴包（Partner Packages）

以下提供者有独立的 `@langchain/{provider}` 包，以提供更好的版本控制、依赖管理和测试支持。

有关如何使用每个包的具体信息，请查阅相应组件文档部分中的页面（例如 [聊天模型](/docs/integrations/chat/)）。
