# ChatPrem

## 安装配置

1. 创建一个 Prem AI 账户，并在此处获取您的 API 密钥 [here](https://studio.premai.io/sign-up).
2. 导出或直接设置您的 API 密钥。ChatPrem 类默认使用 `process.env.PREM_API_KEY`。

```bash
export PREM_API_KEY=your-api-key
```
