# 聊天模型

[聊天模型](/docs/concepts/chat_models) 是一类语言模型，它们使用一系列 [消息](/docs/concepts/messages) 作为输入，并返回消息作为输出（而不是使用纯文本）。这些模型通常是较新的模型。

:::info
如果你想编写自己的聊天模型，请参阅 [本指南](/docs/how_to/custom_chat)。如果你想贡献一个集成，请参阅 [贡献集成](/docs/contributing)。
:::
