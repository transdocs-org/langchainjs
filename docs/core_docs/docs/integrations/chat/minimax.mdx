# Minimax

[Minimax](https://api.minimax.chat) 是一家中国初创公司，为企业和个人提供自然语言处理模型。

此示例演示了如何使用 LangChain.js 与 Minimax 进行交互。

## 准备工作

要使用 Minimax 模型，你需要一个 Minimax 账号、API 密钥以及 Group ID。
