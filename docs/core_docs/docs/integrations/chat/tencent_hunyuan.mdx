# ChatTencentHunyuan

LangChain.js 支持腾讯混元系列模型。

https://cloud.tencent.com/document/product/1729/104753

## 准备工作

1. 注册腾讯云账户 [此处](https://cloud.tencent.com/register)。
2. 创建 SecretID 和 SecretKey [此处](https://console.cloud.tencent.com/cam/capi)。
3. 将 SecretID 和 SecretKey 分别设置为名为 `TENCENT_SECRET_ID` 和 `TENCENT_SECRET_KEY` 的环境变量。

import IntegrationInstallTooltip from "@mdx_components/integration_install_tooltip.mdx";

<IntegrationInstallTooltip></IntegrationInstallTooltip>

```bash npm2yarn
npm install @langchain/community @langchain/core
```

如果你在浏览器环境中使用 LangChain.js，则还需要安装以下依赖：

```bash npm2yarn
npm install crypto-js
```

然后确保你按照下面所示方式从 `web` 中导入。
