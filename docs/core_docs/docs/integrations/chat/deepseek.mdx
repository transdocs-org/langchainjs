## 安装配置

要访问 DeepSeek 模型，你需要创建一个 DeepSeek 账户，获取一个 API 密钥，并安装 `@langchain/deepseek` 集成包。

你也可以通过 [Together AI](../../../docs/integrations/chat/togetherai) 或 [Ollama](../../../docs/integrations/chat/ollama) 等提供商访问 DeepSeek API。

### 凭证

前往 https://deepseek.com/ 注册 DeepSeek 并生成一个 API 密钥。完成之后，请设置 `DEEPSEEK_API_KEY` 环境变量：

```bash
export DEEPSEEK_API_KEY="your-api-key"
```

如果你想获得模型调用的自动追踪，还可以取消注释下面的内容并设置你的 [LangSmith](https://docs.smith.langchain.com/) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

## 实例化

现在我们可以实例化我们的模型对象并生成聊天补全：

```typescript
import { ChatDeepSeek } from "@langchain/deepseek";

const llm = new ChatDeepSeek({
  model: "deepseek-reasoner",
  temperature: 0,
  // 其他参数...
});
```

## 链式调用

我们可以将模型与提示模板进行[链式调用](../../../docs/how_to/sequence/)，如下所示：

```typescript
import { ChatPromptTemplate } from "@langchain/core/prompts";

const prompt = ChatPromptTemplate.fromMessages([
  [
    "system",
    "你是一个能将 {input_language} 翻译为 {output_language} 的有用助手。",
  ],
  ["human", "{input}"],
]);

const chain = prompt.pipe(llm);
await chain.invoke({
  input_language: "English",
  output_language: "German",
  input: "I love programming.",
});
```
