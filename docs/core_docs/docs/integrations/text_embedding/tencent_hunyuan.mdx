# TencentHunyuan

`TencentHunyuanEmbeddings` 类使用腾讯混元 API 为给定文本生成嵌入表示。

## 准备工作

1. 在 [这里](https://cloud.tencent.com/register) 注册腾讯云账号。
2. 在 [这里](https://console.cloud.tencent.com/cam/capi) 创建 SecretID 和 SecretKey。
3. 将 SecretID 和 SecretKey 分别设置为名为 `TENCENT_SECRET_ID` 和 `TENCENT_SECRET_KEY` 的环境变量。

import IntegrationInstallTooltip from "@mdx_components/integration_install_tooltip.mdx";

```bash npm2yarn
npm install @langchain/community @langchain/core
```

```bash npm2yarn
npm install crypto-js
```

然后确保你按照如下方式从 `web` 中导入。
