# Zep Cloud 检索器

> [Zep](https://www.getzep.com) 是一个面向 AI 助手应用的长期记忆服务。
> 使用 Zep，你可以让 AI 助手具备回忆过去对话的能力，无论这些对话多久以前发生过，
> 同时还能减少幻觉、延迟和成本。

此示例展示了如何在检索链中使用 Zep 检索器，从 Zep 开源记忆存储中检索文档。

## 安装

注册 [Zep Cloud](https://app.getzep.com/) 并创建一个项目。

按照 [Zep Cloud Typescript SDK 安装指南](https://help.getzep.com/sdks) 安装并开始使用 Zep。

你需要使用 Zep Cloud 项目的 API 密钥来使用 ZepCloudRetriever。更多信息请参见 [Zep Cloud 文档](https://help.getzep.com/projects)。
