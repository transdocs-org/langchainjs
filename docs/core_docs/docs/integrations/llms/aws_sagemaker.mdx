# AWS SageMakerEndpoint

LangChain.js 支持与 AWS SageMaker 托管端点的集成。有关可用模型列表以及如何部署自己的模型，请查看 [Amazon SageMaker JumpStart](https://aws.amazon.com/sagemaker/jumpstart/)。

## 准备工作

你需要安装官方的 SageMaker SDK 作为同级依赖项：

```bash npm2yarn
npm install @aws-sdk/client-sagemaker-runtime
```
