# 大型语言模型（LLMs）

:::caution
你当前所查看的页面描述的是[文本补全模型](/docs/concepts/text_llms)的使用。目前最新且最流行的模型是[对话补全模型](/docs/concepts/chat_models)。

除非你正在使用更高级的提示技术，否则你可能需要访问[此页面](/docs/integrations/chat/)。
:::

[LLMs](docs/concepts/#llms) 是指接受字符串作为输入，并返回字符串作为输出的语言模型。

:::info
如果你想实现自己的 LLM，请参考[本指南](/docs/how_to/custom_llm)。如果你想为项目贡献集成支持，请查看[贡献集成](/docs/contributing)。
:::
