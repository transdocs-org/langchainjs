## Deeplearning.ai

我们与 [Deeplearning.ai](https://deeplearning.ai) 和 [Andrew Ng](https://en.wikipedia.org/wiki/Andrew_Ng) 合作推出了一个 LangChain.js 短期课程。

该课程涵盖了 LCEL 以及其他可以组合使用来构建更复杂链的组件，同时还介绍了用于检索增强生成（RAG）的数据加载基础知识。你可以通过以下链接免费学习：

- [使用 LangChain.js 构建 LLM 应用](https://www.deeplearning.ai/short-courses/build-llm-apps-with-langchain-js)

## Scrimba 互动指南

[Scrimba](https://scrimba.com) 是一个代码学习平台，允许你在观看视频演示的同时交互式地编辑和运行代码。

我们与 Scrimba 合作开发了一些课程材料（称为 "scrims"），用于教授使用 LangChain.js 构建应用的基础知识 —— 请查看下面的内容，并持续关注更多即将推出的课程！
