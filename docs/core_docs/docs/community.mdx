# 🦜 贡献代码到 LangChain

LangChain 是由 1500 多位贡献者提交的 5000 多个贡献打造而成的，但我们还有**很多**事情可以一起完成。以下是一些参与方式：

- **[打开一个 Pull Request](https://github.com/langchain-ai/langchainjs/issues)：** 我们欢迎任何形式的贡献——新功能、基础设施改进、更好的文档、错误修复等。如果你有一个改进建议或新想法，我们很乐意与你一起完成。
- **[阅读我们的贡献指南：](https://github.com/langchain-ai/langchainjs/blob/main/CONTRIBUTING.md)** 我们要求贡献者遵循 ["fork and pull request"](https://docs.github.com/en/get-started/quickstart/contributing-to-projects) 的工作流程，在提交前运行一些本地检查以确保格式、代码规范和测试通过，并遵循特定的文档和测试规范。
- **成为专家：** 我们的专家会在 Discord 上帮助社区解答产品问题。如果你愿意扮演这个角色，我们将非常感激！（我们还有一些仅限专家的福利/特别待遇可以告诉你）。发送邮件到 hello@langchain.dev 介绍自己，我们后续会联系你！
- **与 LangChain 集成：** 如果你的产品已经或希望与 LangChain 集成，我们希望帮助你和终端用户获得尽可能顺畅的体验。发送邮件到 hello@langchain.dev 告诉我们你正在做什么。
  - **成为集成维护者：** 与我们的团队合作，确保你的集成保持最新状态，并在我们的 Discord 中直接与用户交流（并回答他们的问题）。如果你想探索这个角色，请发送邮件至 hello@langchain.dev 介绍自己。

# 🌍 线下聚会、活动和黑客松

我们在 AI 领域工作时最喜爱的一点就是大家对共同构建项目的热情。我们希望让你们尽可能轻松、高效地参与这些活动！

- **查找线下聚会、黑客松或网络研讨会：** 你可以在我们的 [全球活动日历](https://mirror-feeling-d80.notion.site/0bc81da76a184297b86ca8fc782ee9a3?v=0d80342540df465396546976a50cfb3f) 中找到适合你的活动。
  - **向我们的日历提交活动：** 发送邮件到 events@langchain.dev 并附上你的活动页面链接！我们也可以帮助你在本地社区中宣传。
- **主办线下聚会：** 如果你想组织一群开发者聚会，我们愿意提供帮助！我们可以在我们的活动日历/Twitter 上宣传你的活动，在 Discord 的本地社区中分享，提供周边礼品，甚至可能帮你找到赞助商。发送邮件到 events@langchain.dev 告诉我们你的活动详情！
- **成为线下聚会赞助商：** 我们经常听到一些开发者团体希望组织聚会，但在某些方面受到限制（如场地、零食预算、奖品等）。如果你愿意提供帮助，请发送邮件到 events@langchain.dev，我们可以进一步说明如何操作。
- **在活动中演讲：** 线下聚会的主办方一直在寻找优秀的演讲者、演示者和讨论嘉宾。如果你想参与，请发送邮件到 hello@langchain.dev，介绍你自己、演讲主题以及你所在的城市，我们会尝试为你匹配即将到来的活动！
- **告诉我们你的 LLM 社区：** 如果你组织或参与一个社区，并希望获得 LangChain 和/或我们团队的支持，请发送邮件到 hello@langchain.dev 告知我们如何帮助你。

# 📣 帮助我们宣传你的作品

如果你正在做一个让你自豪的项目，并认为 LangChain 社区可以从了解它中受益，我们愿意帮助你展示它。

- **在社交媒体上发布你的作品并提及我们：** 我们喜欢在 Twitter 上关注领域内的动态。如果你在帖子中@ [@langchainai](https://twitter.com/LangChainAI)，我们几乎肯定会看到它，并可以给予你一些支持。
- **在我们的博客上发表文章：** 如果你正在撰写使用 LangChain 构建项目的体验，我们很乐意在我们的博客上发布（或转载）！发送你的文章草稿到 hello@langchain.dev！甚至只是一个写作想法也可以。
- **将你的产品加入我们的 [集成中心](https://integrations.langchain.com/)：** 许多开发者利用我们与其他产品的无缝集成，并来到我们的集成中心查找这些产品。如果你想让你的产品加入其中，请发送邮件到 hello@langchain.dev，告诉我们你的产品及其与 LangChain 的集成方式。
